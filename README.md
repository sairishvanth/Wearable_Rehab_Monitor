
# Wearable Rehab Monitoring System

A wearable IoT-based posture monitoring system for rehabilitation tracking.

## Overview
Real-time posture monitoring using dual MPU6050 IMU sensors and Raspberry Pi 4, 
with a Flask web dashboard for clinicians to track patient movement data.

## Features
- Dual MPU6050 IMU sensor integration
- Real-time posture angle tracking
- Web-based clinician dashboard (Flask + HTML/CSS/JS)
- Live data visualization

## Tech Stack
- Python
- Raspberry Pi 4
- MPU6050 IMU Sensors
- Flask
- HTML/CSS/JS

## Setup & Installation
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Open browser at `http://<raspberry-pi-ip>:5000`

## Hardware Requirements
- Raspberry Pi 4
- 2x MPU6050 IMU Sensors
- Connecting wires
